# ASGS Webmaster Challenge

## Live Site

https://himir-desai.github.io/ASGS-Webmaster-Challenge/

## Prerequisites

- [Node.js](https://nodejs.org/) v22.12.0 or later

## Getting Started

```bash
cd asce-site
npm install
npm run dev
```

Open the local URL printed in your terminal (usually `http://localhost:4321`).

## Other Commands

| Command | Description |
|---------|-------------|
| `npm run build` | Build for production |
| `npm run preview` | Preview the production build locally |
